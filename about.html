Nossa missão é trazer uma ferramenta de navegação acessível a todos.

Idealizamos uma forma de navegação facilitada para os usuários de carros Ford. Um sistema de GPS em realidade aumentada, algo que já visto antes, mas não desta forma.

Detalhes do projeto

A tela

A captura da estrada feita em tempo real através de uma câmera na parte frontal do carro que fará o mapeamento, assim indicando o caminho de forma interativa e animada.

Tecnologias utilizadas

Passamos por diversas alterações no projeto e finalmente chegamos em algo definitivo. Utilizaremos o Mapbox, uma API feita exclusivamente para renderização das cidades em 3D e navegação de mapa.
